Veröffentlicht am 2017-08-21

windows软件备忘录

系统类

  • seer –> windows快速预览工具
  • wox –> 快速访问类应用
  • ScreenToGif –> 制作gif工具
  • FSCapture –> 截屏录屏工具
  • CCleaner –>垃圾清理工具
  • everything –> 快速搜索文件
  • prime –> 很赞的输入法
  • IDM,Eagle Get –> 都还不错的下载器
  • 深蓝词库工具 –> 制作输入法词库的利器
  • mongoose –> 在网页上显示本地文件
  • ConEmu –> 替代windows自带的终端，看起来很赞
  • Far Manager –> 文件管理器 配合ConEmu一起用

阅读编辑类

  • sublime text –> 编写代码最好的工具
  • Simplenote –> 特别简洁，可以用来记录日常感悟心得
  • OneNote –> 大量资料整理归类很有用
  • Notepad ++ –> 日常处理文档非常高效好用
  • Xodo (UWP) –> 查看pdf不错
  • Code Writer(UWP) –> 还不错
  • Bamboo Paper(UWP) –> 用于触屏的笔记本
  • Texmaker –> LaTeX 可視化編輯器

播放器类

  • Potplayer –>一直以来的最爱
  • ACG播放器 –> 视觉效果很赞
  • 网易云播放器 –> 一直很好用
  • 爱奇艺 –> 很好
  • 芒果tv –> 还不错

英语相关

  • Aboboo –> 学习英语的利器
  • GoldenDIct –> 很棒的英语翻译软件
  • QTranslate –> 很棒的划译软件

爱好相关

* Sabaki –> 很美观的围棋软件

还不错的网址

  • 观察者网 –> 非常‘红’的网站
  • weavi –> 在线写作的网站，可以记零碎话
  • 历史的今天 –> 历史的今天，每天起床可以看一看
  • raindrop –>
